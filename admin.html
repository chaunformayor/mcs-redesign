Change Admin Password

Edit admin.php on your server and update the ADMIN_PASSWORD constant.

define('ADMIN_PASSWORD', 'your-new-password');

Change Site Password

Edit siteprotect.php on your server and update the SITE_PASSWORD constant. To disable site protection, remove the 5 TEMP lines from .htaccess.

define('SITE_PASSWORD', 'your-new-password');
